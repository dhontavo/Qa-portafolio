# PBTimeShare, PBAdmin, PBResortManager (Reingeniería)
Este proyecto incluye las pruebas manuales realizadas dentro de las 3 aplicaiónes web.
El objetivo del proceso de pruebas fue validar la funcionalidad. validación de datos y el comportamiento general tanto de la aplicación web.

Las actividades de prueba incluyeron pruebas funcionales, pruebas de validacion, pruebas de interfaces.

------------


## Web Application Testing

Las pruebas se centraron en validar las funcionalidades claves y datos del formulario.

### Activities performed
- Pruebas funcionales mediante listas de verificación
- Validación de los campos de entrada del usuario
- Identificación de defectos
### Artifacts
El documento incluye:
- Realización de checklist de pruebas
- Fomato de validación de pruebas
